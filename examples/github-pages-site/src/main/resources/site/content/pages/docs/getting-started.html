Installation

To get started with ROQ, add the following dependency to your project:

<dependency>
    <groupId>io.quarkiverse.roq</groupId>
    <artifactId>quarkus-roq</artifactId>
</dependency>

Base Path Configuration

Configure your base path for different environments:

# application.properties
%dev.site.base-path=
%prod.site.base-path=/my-project

This allows your site to work both locally (no base path) and in production (with base path).

Template Usage

Use the base path in your templates:

<!-- Asset paths -->
<link rel="stylesheet" href="{site.assetPath('/css/style.css')}">

<!-- Navigation links -->
<a href="{site.basePath}/docs/">Documentation</a>
