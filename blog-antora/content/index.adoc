= Quarkus Roq

Quarkus Roq is a static site generator that leverages Quarkus to build blogs, documentation sites, and static websites with ease.

For full documentation, visit the https://iamroq.com/docs/[Roq website].

== Core Extensions

Roq consists of several extensions that can be used independently:

* link:/quarkus-roq-data/[*Roq Data*] - Access JSON/YAML data files from templates with type safety
* link:/quarkus-roq-frontmatter/[*Roq FrontMatter*] - Build websites from Markdown/AsciiDoc/HTML pages with frontmatter headers
* link:/quarkus-roq-generator/[*Roq Generator*] - Extract any Quarkus web application as pure static files
