You can access page data like this:
```markdown
* \{page.data.cool}
* \{page.data.fun}
```
**will render ⤵**

* {page.data.cool}
* {page.data.fun}


There are a few helpers on the `page` variable ([more on variables](https://docs.quarkiverse.io/quarkus-roq/dev/index.html#variables)):

```markdown
> \{page.date.format('YYYY')}: \{page.description}
```
**will render ⤵**

> {page.date.format('YYYY')}: {page.description}
